Atrações Principais

O Evento Javeiros é a maior conferência de desenvolvedores Java da América Latina. Neste ano, você pode esperar workshops práticos, palestras inspiradoras e muita oportunidade para networking. Abaixo, confira as principais atrações:

  • Keynote de Abertura - "O Futuro do Java": Uma palestra imperdível com os principais desenvolvedores e líderes de comunidades Java.
  • Workshop: Desenvolvendo Microservices com Spring Boot: Aprenda na prática como construir e implantar microservices com Spring Boot, um dos frameworks mais utilizados no mundo Java.
  • Painel: Oportunidades de Carreira para Desenvolvedores Java: Especialistas da indústria discutem como evoluir na carreira de desenvolvedor, desde júnior até arquiteto de software.
  • Hands-on: Integração de Java com Inteligência Artificial: Descubra como usar Java para construir soluções inteligentes utilizando bibliotecas de IA.
  • Hackathon Java: Teste suas habilidades em programação Java em uma maratona de desenvolvimento com prêmios exclusivos!
